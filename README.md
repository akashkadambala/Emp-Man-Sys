# Employee Management System

A simple yet effective **Employee Management System** built to demonstrate core concepts of Jsp's, Java servlets, database integration, and CRUD operations. This project is designed for learning, practice, and showcasing recruiter‑ready skills in full‑stack development.

---

## 🚀 Features
- Add, update, and delete employee records
- View employee details in a structured format
- Search and filter employees by attributes
- Modular codebase for easy extension


---

## ⚙️ Setup Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/akashkadambala/Emp-Man-Sys.git
   cd Emp-Man-Sys
2. Configure the database
   Import the SQL schema and run it on Mysql Workbench.
3. Use any Java dedicated IDE(Eclipse) to open the java files.
4. Install Apache Tomcat to run Jsp files in localhost server.
5. Configure Your own pom.xml file for this project.
